# 🌌 Awesome OS Wallpapers

A curated collection of high-quality wallpapers for your desktop, organized by categories. This repository contains over 600+ wallpapers totaling ~3GB of stunning visuals.

## 🚀 Usage / Integration

You can use these wallpapers directly in your projects via the JSDelivr CDN.

### 1. HTML
```html
<img src="https://cdn.jsdelivr.net/gh/Tcode-Motion/os-wallpapers/wallpapers/abstract/wallhaven-135529.jpg" alt="Wallpaper">
```

### 2. CSS (Background)
```css
body {
  background-image: url("https://cdn.jsdelivr.net/gh/Tcode-Motion/os-wallpapers/wallpapers/abstract/wallhaven-135529.jpg");
  background-size: cover;
}
```

### 3. JavaScript
```javascript
const wallpaperUrl = "https://cdn.jsdelivr.net/gh/Tcode-Motion/os-wallpapers/wallpapers/abstract/wallhaven-135529.jpg";
document.body.style.backgroundImage = `url(${wallpaperUrl})`;
```
